# Ensemble Learning Project

## Overview
This project compares multiple ensemble learning algorithms for classification.

## Models Used
- Gradient Boosting
- AdaBoost
- XGBoost
- Voting Classifier

## Technologies
- Python
- Pandas
- NumPy
- Scikit-Learn
- XGBoost


## Results

| Model | Accuracy |
|--------|----------|
| Gradient Boosting | 79.69% |
| AdaBoost | 81.45% |
| XGBoost | 77.64% |


Best Performing Model: XGBoost

## How to Run

pip install -r requirements.txt

Run:
Ensemble_Project.ipynb
